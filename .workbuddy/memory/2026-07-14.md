# 2026-07-14 工作日志

## 修复 send_md_email.py 邮件发送问题

**问题原因（3个）：**
1. **附件路径错误** — 脚本中 `md_file = "output/weekly-2026-07-13.md"`，但实际文件路径是 `output/weekly-2026-07-13/newsletter.md`（目录下的 newsletter.md）
2. **缺少依赖包** — yagmail 依赖 `premailer` → `cssutils` → `more-itertools`，但环境中未安装 `more-itertools`，导致 `ModuleNotFoundError`
3. **SMTP 连接参数不完整** — 原代码未指定 `port` 和 `smtp_ssl`，QQ邮箱需要 `port=465, smtp_ssl=True`

**修复内容：**
- 修正附件路径为 `output/weekly-2026-07-13/newsletter.md`
- 添加 `os.chdir` 确保相对路径正确
- 补充 `port=465, smtp_ssl=True` 参数
- 安装缺失依赖：`more-itertools`, `cssutils`, `premailer` 及其传递依赖

**结果：** 邮件发送成功。
