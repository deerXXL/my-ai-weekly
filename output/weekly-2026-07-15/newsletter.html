🚀 行业动态

7月15·周三

• 阿尔伯塔省政府部署Claude修复网络安全漏洞

Anthropic发布案例研究显示，加拿大阿尔伯塔省政府正利用Claude在其庞大的政府系统中自动发现并修复网络安全漏洞。通过将大模型嵌入安全运营流程，政府得以加快漏洞识别与响应速度，降低敏感数据风险。此举展示了大模型在公共部门网络安全领域的落地价值，也为其他政府机构采用AI提升安全能力提供了参考范例。

使用说明：政府IT团队可将Claude接入安全运营流程，自动扫描系统漏洞并生成修复建议，加速响应

• Hugging Face 一键部署至 Amazon SageMaker Studio

亚马逊与 Hugging Face 合作推出一键部署功能，用户可直接从 Hugging Face 模型页面无缝跳转至 Amazon SageMaker Studio 进行训练和部署，大幅简化了开源模型在 AWS 云端的接入流程。此举将降低企业使用开源模型的门槛，进一步推动开源生态与云平台的深度整合。

使用说明：开发者在HuggingFace模型页一键跳转SageMaker Studio，即可完成开源模型的训练与部署

7月14·周二

• 微软Foundry托管算力上线Hugging Face模型

微软在Hugging Face发布博客，宣布其Azure AI Foundry托管算力平台正式支持Hugging Face生态中的开源模型部署。开发者可通过Foundry的托管计算基础设施，一键部署和运行Hugging Face上的各类模型，享受企业级算力、安全与可扩展性。此举进一步深化了微软与Hugging Face的合作，降低了开源大模型在企业场景中的落地门槛。

使用说明：企业用户可通过Azure AI Foundry托管算力直接部署HuggingFace开源模型，享受企业级安全与扩展性

• HuggingFace 发布 LeRobot v0.6.0：想象、评估与改进

HuggingFace 正式推出开源机器人框架 LeRobot v0.6.0 版本，围绕"想象、评估、改进"三大主题升级机器人学习流程。新版本强化了策略训练、仿真评估和模型迭代能力，为研究者与开发者提供更完善的端到端工具链，进一步推动具身智能与机器人学习的开源普及，加速社区在真实世界机器人任务上的探索与落地。

使用说明：机器人研究者可用LeRobot v0.6.0完成策略训练、仿真评估到模型迭代的端到端工作流

7月13·周一

• 英伟达发布智能体开放数据集，助力Agent训练

英伟达在Hugging Face发布面向AI智能体的开放数据集Data for Agents，为训练和评估自主智能体提供高质量语料。该数据集涵盖多样化的任务场景与交互轨迹，旨在降低Agent研发门槛，推动开源社区构建更强大的推理与工具调用能力，标志着智能体训练数据生态迈出重要一步。

使用说明：Agent开发者可使用该数据集训练具备工具调用与推理能力的自主智能体，降低语料准备成本

• vLLM原生集成Transformers后端，推理速度接近原生

HuggingFace宣布vLLM推理框架已实现对Transformers建模后端的原生速度支持。此举意味着开发者可直接使用Transformers库中的模型定义，无需额外适配即可在vLLM上获得接近原生优化的推理性能。这大幅降低了新模型部署到高性能推理服务的门槛，加速了开源模型从训练到生产的落地流程，进一步巩固了HuggingFace与vLLM在开源推理生态中的协同地位。

使用说明：开发者可直接用Transformers模型定义在vLLM上部署，无需重写即可获得接近原生的推理速度

全球数字化转型解决方案商UST宣布与Anthropic合作，将Claude大模型引入其物理AI（Physical AI）业务，赋能机器人、工业自动化及智能设备等实体场景。该案例展示了大语言模型从纯软件走向物理世界的落地路径，标志着Claude在企业级具身智能和工业智能化领域的应用拓展，为制造业和物流等行业的AI转型提供新范式。

前美联储主席本·伯南克被任命为Anthropic长期利益信托（LTBT）成员。该信托是Anthropic独特治理架构的核心，负责监督公司使命的长期落实，确保AI安全与公共利益优先于股东回报。伯南克的加入为Anthropic带来了顶级宏观经济和政策治理经验，有助于其在AI监管日益复杂的环境中稳健发展。

使用说明：K-12及高校教师可用教师版Claude进行备课、设计课程、批改作业和生成教学材料

使用说明：求职者可本地部署该框架，用Claude Code完成岗位评估、简历定制、求职信撰写和面试准备

7月10·周五

• OfficeCLI开源：首个专为AI智能体打造的Office套件

GitHub上开源项目OfficeCLI发布，号称首个专为AI智能体设计的Office命令行套件，支持读取、编辑和自动化处理Word、Excel、PowerPoint文件。工具以单一二进制文件形式提供，无需安装Microsoft Office即可运行，为AI Agent处理办公文档场景提供了轻量化基础设施，有望推动办公自动化与智能体生态融合。

使用说明：AI Agent通过OfficeCLI以命令行方式读写Word、Excel、PPT，无需安装Office即可实现办公自动化

• Stably AI 推出 Orca：并行智能体舰队开发环境

Stably AI 在 GitHub 开源了 Orca，一款面向并行智能体舰队的智能体开发环境（ADE）。开发者可使用自己的订阅账户运行任意编码智能体，并通过桌面端和移动端进行统一管理与调度。Orca 意在解决多智能体协同开发中的编排难题，为大规模并行代码生成场景提供基础设施，推动智能体工程化落地。

使用说明：开发者用自有订阅账号在Orca中并行调度多个编码智能体，通过桌面与移动端统一管理任务

GitHub开源项目DesktopCommanderMCP是一款面向Claude的MCP服务器，可为其提供终端命令执行、文件系统搜索及差异化文件编辑等能力。该工具让Claude突破对话边界，直接操作本地开发环境，显著增强了AI在编程、自动化和系统运维场景下的实用性，是MCP生态快速扩张的又一例证。

使用说明：Claude用户接入DesktopCommanderMCP后，可直接执行终端命令、搜索文件和进行差异化编辑

阶跃星辰推出首款大模型原生智能体手机STEPX Neo，搭载自研智能体原生操作系统Step AOS，内置系统级个人智能体Amoo。用户可通过自然语言指令实现跨应用自主调度，系统具备长期记忆和端云协同能力，支持灵活任务路由。首批生态合作伙伴涵盖支付宝、美团、高德、滴滴、京东、百度等主流应用，标志着AI原生手机进入实用化阶段。

使用说明：用户通过自然语言指令即可跨支付宝、美团、高德等应用调度任务，享受端云协同的智能体体验

7月8·周三

• OpenAI发布GPT-5.6系列：Sol、Terra、Luna三档齐发

OpenAI正式推出GPT-5.6大模型系列，包含旗舰版Sol、均衡版Terra和性价比版Luna三款产品，API定价覆盖每百万token 1至30美元的梯度区间。在Agents' Last Exam评测中，Sol以53.6分领先Claude Fable 5达13.1分，编程智能体指数创下80分新高，同时在成本与响应耗时上均实现大幅优化，进一步巩固OpenAI在大模型竞赛中的领先地位。

使用说明：开发者可根据成本与性能需求，在Sol、Terra、Luna三档间灵活选择API调用，覆盖不同场景

OpenAI 发布全新智能体产品 ChatGPT Work，由 Codex 与 GPT-5.6 驱动，定位为可执行长时间、多步骤任务的智能体，覆盖网页、移动端和桌面平台。用户仅需一句指令描述目标，AI 便可接管整个工作流，结合应用和文件上下文自动生成文档、幻灯片、分析报告和网站，标志着 OpenAI 正加速向企业级智能体办公场景延伸。

使用说明：用户一句指令描述目标，ChatGPT Work可跨网页、移动端和桌面自动生成文档、幻灯片和分析报告

7月7·周二

• OpenAI发布新一代语音模型GPT-Live，支持全双工实时对话

OpenAI推出全新语音交互模型GPT-Live，支持实时同声传译与全双工对话，用户可随时插话打断。该模型允许自定义推理强度，并引入深度任务委托机制：前台保持流畅语音交互的同时，后台可并行执行联网搜索、复杂推理等任务，并在对话中实时弹出天气、赛事等可视化卡片，标志着语音AI进入多模态并行新阶段。

使用说明：用户可用GPT-Live进行实时同传或全双工对话，边说边看AI在后台生成的天气、赛事等卡片

• 阿里元境推出JellyToken：一站式聚合60余款国产大模型

阿里元境正式发布AI大模型聚合平台JellyToken（智渲云），用户通过单一API Key即可调用通义千问、DeepSeek、豆包、智谱、月之暗面等60余款国产模型，覆盖文本、图像、视频、音频四大场景。平台配备智能路由、负载均衡、Token级精准计费及正规发票等企业级功能，显著降低多模型接入与运维成本，加速国产大模型在企业侧的规模化落地。

使用说明：企业开发者用单一API Key即可调用通义、DeepSeek、豆包等60余款国产模型，覆盖文本、图像、视频、音频
