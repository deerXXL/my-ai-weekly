闪联AI周刊

🗓 本周概览

时间范围：2026年7月1日 - 2026年7月14日

本期编辑：产品资讯组

核心摘要：近两周AI行业进入模型迭代与生态争夺的高强度阶段。OpenAI一周内密集发布GPT-5.6三档模型、ChatGPT Work智能体工作台与GPT-Live全双工语音，正式将战场从对话推向企业级自动化与实时多模态交互；Anthropic则以Claude Code、Fable 5安全框架与升级版负责任扩展政策巩固安全叙事，并借伯南克入局与卢比计价加速治理公信力和新兴市场布局。国内侧，蚂蚁开源双安全模型、阿里元境上线国产大模型聚合平台JellyToken、智源发布RoboBrain Orca世界模型、阶跃星辰推出Step Edge端侧系列，从Agent安全、模型分发到具身与端侧全线推进。同时，欧洲Soofi S开源大模型、深圳机器人独角兽百亿融资，也标志着AI主权与具身智能成为新一轮资本与技术焦点。

🚀 行业动态

7月14·周二

• Anthropic披露Fable 5网络安全防护与越狱评估框架

Anthropic发布公告，详细介绍其新一代模型Fable 5的网络安全防护措施，并公开了用于评估模型抗越狱能力的系统性框架。此举旨在提升前沿模型在面对恶意攻击和滥用时的稳健性，为行业在AI安全评测标准化方面提供参考，也回应了外界对大模型安全性和可控性的持续关切。

使用说明：面向AI安全研究者与红队，可参考其越狱评估框架搭建自有模型的抗攻击测试体系。

7月13·周一

• Anthropic揭秘Claude Code诞生历程

Anthropic发布长文，回顾其编程智能体Claude Code从一款内部CLI工具演变为正式产品的全过程。文章由参与研发的研究员、工程师及早期用户共同讲述，披露了产品的设计理念、迭代路径与内部使用经验。Claude Code已成为Anthropic在编程Agent赛道对标Cursor、GitHub Copilot的核心武器，此次公开幕后故事有助于外界理解其技术路线与竞争定位。

使用说明：开发者可借鉴Claude Code的CLI式Agent设计思路，用于构建代码助手与自动化编程工作流。

• 阿尔伯塔省政府用Claude排查政府系统安全漏洞

Anthropic发布案例研究显示，加拿大阿尔伯塔省政府正利用Claude在其政府IT系统中发现并修复网络安全漏洞。通过AI辅助的漏洞识别与修复流程，政府部门显著提升了安全审计效率，为公共部门大规模应用AI进行网络安全治理提供了参考样本，也进一步彰显Anthropic在政企合规场景的拓展。

使用说明：适用于政企IT团队，可将Claude接入代码审计与漏洞扫描流程，提升安全运维效率。

7月12·周日

• UST携手Claude落地物理AI，工业智能化提速

数字化转型服务商UST宣布将Anthropic的Claude模型引入其"物理AI"解决方案，用于工厂、供应链等实体工业场景。通过将大模型能力与机器人、传感器和自动化设备结合，Claude将协助完成质量检测、流程优化和现场决策。此次合作标志着通用大模型从数字世界向物理世界延伸，为制造业智能化和工业AI落地提供了新范例。

使用说明：制造业可将Claude结合机器人与传感器，用于工厂质检、供应链决策与产线优化。

• Anthropic 推出 Claude 使用反思功能

Anthropic 发布了一项新功能，帮助用户追踪并可视化自己使用 Claude 的方式，并评估这些时间投入是否契合个人目标。该功能旨在提升用户对 AI 使用习惯的自我认知，推动更有意识、更负责任的人机互动，也反映出 AI 公司开始关注用户数字健康和使用效率。

使用说明：普通ChatGPT/Claude用户可通过反思面板查看AI使用时长与场景分布，优化个人使用习惯。

7月11·周六

• Anthropic征集AI难题：承诺公开回应公众质疑

Anthropic发起面向公众的征集活动，邀请人们提出关于AI最棘手的问题，涵盖安全、伦理、社会影响等敏感议题。公司承诺将公开展示其思考过程和回应工作，以透明方式面对外界质疑。此举体现了头部AI公司主动接受社会监督的姿态，也是行业信任建设的重要尝试，有助于推动AI治理讨论走向深入。

• 伯南克加入Anthropic长期利益信托

Anthropic宣布前美联储主席本·伯南克加入其长期利益信托委员会（LTBT）。该信托是Anthropic独特治理架构的核心，负责监督公司使命的履行，并有权任命部分董事会成员。伯南克在宏观经济与政策监管领域的深厚经验，将助力Anthropic在AI安全与社会影响层面的长期决策，凸显公司加强治理公信力的意图。

7月10·周五

• GitHub Trending 数据暂时不可用

GitHub 官方接口暂时无法访问，导致本次未能成功抓取 GitHub Trending 榜单数据。此为临时性技术故障，不涉及具体项目动态，读者可稍后自行访问 GitHub 官网查看最新热门开源项目。

• 德国研究联盟推出开源大模型Soofi S，强化欧洲AI主权

德国研究联盟发布开源大模型Soofi S30B-A3B，助力欧洲AI主权建设。该模型采用混合专家(MoE)架构，融合Mamba-2与注意力层，总参数量达316亿，但推理时每个token仅激活部分参数，兼顾高性能与高效生成。此举为欧洲在开源大模型领域争取话语权，减少对美国科技巨头的依赖，为全球开源生态注入新活力。

使用说明：欧洲开发者与研究机构可基于Soofi S30B-A3B进行本地化微调，用于对数据主权敏感的场景。

7月9·周四

• 需求暴增，OpenAI临时解除GPT-5.6 Sol算力限制

面对GPT-5.6 Sol在48小时内的爆发式需求增长，OpenAI宣布临时取消Plus、Pro和Business用户的5小时使用限制，并重置所有用户的一次性额度，以保障Codex与ChatGPT Work等编程和智能体工作流的连续运行。此举反映出旗舰模型在高强度生产场景中的算力压力，OpenAI同时表示正在对GPT-5进行全面优化以应对持续增长的调用需求。

• 蚂蚁集团开源双安全模型 为大模型智能体戴上"紧箍咒"

蚂蚁AI安全实验室连续开源两款安全防护模型：面向自主执行智能体的安全护栏SingGuard-NSFA，以及应对多模态交互大模型风险的SingGuard。此举标志蚂蚁在AI安全领域完成系统化布局，将防护范围从传统的模型输出审核，延伸至智能体行为控制、权限管理与系统治理层面，为行业智能体落地提供开源安全基础设施。

使用说明：Agent开发者可集成SingGuard系列作为智能体行为护栏，防范多模态越权与滥用风险。

7月8·周三

• OpenAI发布GPT-5.6系列：Sol旗舰模型登顶智能体评测

OpenAI正式推出GPT-5.6系列，包含旗舰Sol、均衡款Terra及性价比Luna三款模型，API定价覆盖每百万token $1至$30。在Agents' Last Exam评测中，Sol以53.6分领先Claude Fable 5达13.1分，编程智能体指数创下80分新高，同时大幅降低成本与响应耗时，进一步巩固OpenAI在智能体和编程赛道的领先地位。

使用说明：Sol适合复杂智能体与高阶编程，Terra面向常规业务，Luna适用于成本敏感的大规模调用。

• OpenAI发布ChatGPT Work智能体工作台

OpenAI推出全新智能体产品ChatGPT Work，由Codex与GPT-5.6驱动，定位为可承担长时间、多步骤复杂任务的智能体，支持网页、移动和桌面全平台协同。用户只需一句指令描述目标，AI便可接管整个工作流程，结合应用与文件上下文自动生成文档、幻灯片、数据分析、网站及报告，标志着OpenAI正式发力企业级自动化办公场景。

使用说明：企业员工可用一句话指令让ChatGPT Work自动完成文档、PPT、数据分析等多步骤办公任务。

7月7·周二

• OpenAI发布GPT-Live语音模型，支持全双工实时对话

OpenAI推出新一代语音交互模型GPT-Live，具备实时同声传译与全双工对话能力，用户可随时插话打断。模型支持自定义推理强度，并引入深度任务委托机制，前台保持流畅语音交互的同时，后台可并行执行联网搜索、复杂推理等任务，还可在对话中实时弹出天气、赛事等可视化卡片，进一步推动语音AI向多模态实用化演进。

使用说明：适合客服、同传、语音助理场景，全双工可打断特性显著提升人机语音交互自然度。

7月6·周一

• 阿里元境发布JellyToken：一站式聚合60余款国产大模型

阿里元境正式推出AI大模型聚合平台JellyToken（智渲云），开发者仅需一个API Key即可调用通义千问、DeepSeek、豆包、智谱、月之暗面等60余款国产大模型，覆盖文本、图像、视频、音频四大场景。平台提供智能路由、负载均衡、Token级精准计费及企业发票等能力，显著降低多模型接入门槛与运维成本。

使用说明：开发者一个API Key即可调用60+国产大模型，适合多模型对比测试与生产环境路由分发。

• 智源发布悟界·RoboBrain Orca 多模态世界模型

智源研究院正式推出悟界·RoboBrain Orca多模态表征世界模型。该模型利用12.5万小时视频进行无意识学习，并结合1.6亿条事件标注开展有意识学习，构建统一的世界潜在表征空间。在文本生成、图像预测及具身动作生成等下游任务中，其表现均优于同等规模的基线模型，为具身智能研究提供了新思路。

使用说明：具身智能研究团队可基于Orca统一表征空间开展视频理解、动作生成与世界建模研究。

7月5·周日

• Anthropic 推出卢比计价，正面挑战 OpenAI 印度市场

7月13日，Anthropic 在印度推出 Claude 本地化卢比计价服务，覆盖官网及移动应用。Claude Pro 月费2000卢比（约21美元），Max 起价11999卢比，团队套餐每席2399卢比。印度已成为 Anthropic 全球第二大市场，用户占比5.8%。此举意在抢占 OpenAI 深耕的印度腹地，加速本地化布局，全球 AI 大厂新兴市场争夺战进一步升级。

• 阶跃星辰推出Step Edge端侧多模态模型系列

阶跃星辰发布面向手机、车载等终端场景的Step Edge系列模型，涵盖基础版、Audio版、GUI版和Gen版，支持本地图文音频处理，具备屏幕理解、语音识别、界面操作与图像生成能力，工具调用延迟低至0.1秒。该系列采用端云协同架构，简单高频任务本地完成，复杂推理交由云端，为端侧多模态AI落地提供新方案。

使用说明：手机、车载与IoT厂商可集成Step Edge实现端侧屏幕理解、语音操作与图像生成能力。

  • 语音交互： GPT-Live全双工可打断特性适合客服与实时助理
